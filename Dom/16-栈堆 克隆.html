如果引用的是引用数据类型（number string boolean undefined null），则后面变量修改后，前面的赋值也跟着改变 因为栈中地址指的是堆中同一块元素 var person1={ name:"zhangsan"} var person2=person1； person1.name='lisi'; console.log(person2.name); 输出结果为lisi 如果引用的是基本数据类型（array.object对象），则在后面修改前面的变量后，赋值不变。仍为前面的 var person1={ name:"zhangsan"} var person2.name =person1.name；person1.name='lisi'; console.log(person2.name); 输出结果为zhangsan
